# Universal Phase Retrieval Transformer for Single-Pattern Structured Light Three-Dimensional Imaging

## Dependencies
- Python 3.9
- PyTorch 1.10.0

```
cd code
pip install -r requirements.txt
python setup.py develop
```
## Datasets

### Train

```shell

```
